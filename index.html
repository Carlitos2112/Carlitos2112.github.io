Manual de Usuario del Geoportal

Bienvenido al manual de usuario de nuestro geoportal. Aquí encontrarás toda la información necesaria para utilizar eficientemente nuestra plataforma.

Cómo acceder al geoportal

Para acceder al geoportal, sigue estos pasos:

  1. Abre tu navegador web preferido.
  2. Ingresa la URL: www.tugeoportal.com
  3. Si es necesario, introduce tus credenciales de acceso.
Ir al Geoportal

Interfaz de usuario y navegación básica

La interfaz del geoportal está diseñada para ser intuitiva y fácil de usar. Aquí te explicamos los elementos principales:

  • Barra de navegación superior: Contiene enlaces a las secciones principales.
  • Panel de capas: Permite activar y desactivar diferentes capas de información geográfica.
  • Área de mapa: Muestra la información geográfica seleccionada.
  • Herramientas de zoom: Permiten acercar y alejar la vista del mapa.
