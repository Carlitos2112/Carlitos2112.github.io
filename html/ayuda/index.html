Manual de Usuario del Geoportal

Bienvenido al manual de usuario de nuestro geoportal. Aquí encontrarás toda la información necesaria para utilizar eficientemente nuestra plataforma.

Interfaz de usuario y navegación básica

La interfaz del geoportal está diseñada para ser intuitiva y fácil de usar. Aquí te explicamos los elementos principales:

1. Herramienta de trazado de polígonos

Esta herramienta permite habilitar la opción para trazar el polígono de nuestra área de interés. Utilizando esta función, podrás delinear específicamente la región que deseas analizar en el mapa.

2. Selección del área de interés

Una vez que hayas hecho clic en el botón "Dibujar Área", aparecerán dos opciones en la pantalla. Debes seleccionar la opción que está encerrada en un cuadrado de color rojo para comenzar a definir tu área de interés.

3. Ingreso de fechas

Después de dibujar el área, debes ingresar la fecha de inicio y final del periodo que deseas evaluar. Asegúrate de usar el formato correcto para las fechas: (YYYY-MM-DD). Esto permitirá que el sistema sepa exactamente qué intervalo de tiempo deseas analizar.

A continuación, te mostramos un ejemplo de cómo ingresar las fechas correctamente en el formato requerido (YYYY-MM-DD).

4. Selección del tipo de índice

Con las fechas correctamente ingresadas, el siguiente paso es seleccionar el tipo de índice que deseas utilizar para tu análisis. Podrás elegir entre índices normalizados y no normalizados. Si aparecen ambas opciones, selecciona la que más se ajuste a tus necesidades.

5. Procesamiento de datos

Una vez que hayas completado todos los pasos anteriores, solo te quedará presionar el botón con la leyenda "Procesar". Este botón iniciará el proceso basado en las configuraciones que seleccionaste anteriormente, y el sistema comenzará a analizar el área y el periodo que especificaste.
